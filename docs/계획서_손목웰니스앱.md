# 프로젝트 계획서 — 데일리 손목 웰니스 앱

> **현대인의 고질적인 손목 피로를 해결하기 위해, 접근성 높은 숏폼 가이드와 몰입감 있는 미니게임을 결합하여, 바쁜 일상 속에서 누구나 쉽고 재미있게 손목 건강을 지킬 수 있도록 돕는 데일리 손목 웰니스 루틴 앱**

---

## 1. 개요

| 항목 | 내용 |
|---|---|
| 프로젝트명 | 데일리 손목 웰니스 앱 (가칭) |
| 형태 | 웹앱 (브라우저 실행, 설치 불필요) |
| 핵심 기술 | 웹캠 + AI 손 인식(MediaPipe), Canvas, 로컬 데이터 저장 |
| 포지셔닝 | 개인용 건강관리(웰니스) — 질병의 진단·치료 목적 아님 |
| 목표 | **"이 앱 하나로 손목을 컨트롤한다"** — 운동 가이드 · 측정 · 기록 · 게임을 하나로 |

## 2. 배경 및 문제 정의

**손목 부담은 이미 통계로 확인되는 사회적 현상이다**
- 건강보험심사평가원 분석에 따르면 손목터널증후군 진료인원은 연평균 약 9%씩 꾸준히 증가해 왔으며(2009년 약 12.4만 명 → 2013년 약 17.5만 명, 총진료비 약 396억 원), 이후에도 증가 추세가 이어지고 있다
- 원인으로 장시간 컴퓨터 작업, 손목을 구부린 자세, 손의 빠른 반복 사용 등이 지목된다 — 즉 **키보드·마우스·스마트폰 중심의 현대적 생활양식 그 자체가 위험 요인**이다
- 질환 진단까지 가지 않더라도, 장시간 디지털 기기 사용자 대부분이 손목 뻐근함·피로를 일상적으로 경험한다

**그런데 손목은 "관리의 사각지대"다**
- 목·허리는 스트레칭 콘텐츠와 인식이 어느 정도 자리 잡았지만, 손목은 **사용량 대비 관리 습관이 가장 없는 부위**
- 스트레칭이 도움이 된다는 것은 알지만 ① 지루하고 ② 정확한 방법을 모르고 ③ 잊어버림
- 병원·전문 관리는 건강한 일반인의 "피로" 수준에는 과하고 시간·비용 부담이 큼
- 영상 콘텐츠(유튜브 등)는 일방향이라 내가 제대로 하는지 피드백이 없고, 기록이 남지 않아 꾸준함으로 이어지지 않음

**왜 지금인가**
- 스마트폰·PC 사용 시간은 계속 늘고, 공부·업무·여가가 모두 화면 앞에서 이뤄지는 환경이 고착됨 → 손목 부담은 구조적으로 증가
- 반면 웹캠·온디바이스 AI(브라우저 손 인식)가 보편화되어, **별도 장비 없이 집에서 동작을 인식·기록하는 서비스가 기술적으로 가능해진 시점**

**핵심 통찰**
- 필요한 것은 정밀한 의료가 아니라, **짧고 · 쉽고 · 재미있고 · 기록되는** 일상 관리 도구
- 문제의 본질은 "방법"이 아니라 **"지속(루틴화)"** — 그래서 해법도 콘텐츠가 아니라 **루틴 형성 시스템**이어야 한다

## 3. 목적 및 타겟

**목적**
- 손목의 **피로 완화 스트레칭**과 운동을 짧은 가이드로 누구나 쉽게 따라 하게 한다
- **홈트레이닝 · 휴식 · 릴렉스 · 운동 가이드**를 한 앱에서 제공한다
- ROM(관절가동범위) 측정 데이터를 **저장·기록·비교**하여 내 손목 상태 변화를 확인하게 한다
- **게임을 보조 수단으로** 활용해 지루하지 않게, 데일리 루틴(습관)을 형성하게 한다

**타겟**
- **컴퓨터·스마트폰을 많이 사용하는 사람 (직장인, 학생)**
- **손목 보호가 필요한 건강한 일반인** — 피로 개선이 목적 (환자 대상 아님)

**사용자 장면 (페르소나)**
- 하루 8시간 모니터 앞에서 일하는 직장인: 오후만 되면 손목이 뻐근하지만 병원 갈 정도는 아니라 그냥 참는다 → **업무 틈 2분, 화면 앞에서 그대로** 가이드를 따라 하고 기록이 쌓인다
- 과제·게임으로 키보드를 오래 쓰는 대학생: 스트레칭 영상을 찾아본 적은 있지만 3일을 못 갔다 → **게임처럼 재미가 있으니** 매일 이어진다

## 4. 핵심 기능 (3축)

### ① 숏폼 운동 가이드 — 메인 🎬
- 손목 운동·스트레칭을 **짧은 영상/애니메이션 자료**로 보여주고 **보고 따라 하게** 함
- **시간은 간단하게, 짧게** (1~3분 루틴) — 바쁜 일상 속 부담 없는 숏폼 형식
- 운동 구성은 문헌(논문) 근거 기반으로 전문성 확보
- 자료는 자체 제작(애니메이션/직접 촬영)으로 저작권 문제 원천 차단
- 각 가이드에 "통증이 느껴지면 즉시 중단" 안내 포함

### ② ROM 측정 & 기록 — 데이터 📊
- 웹캠 + AI 손 인식으로 손목 가동범위를 별도 장비 없이 자가 측정
- **측정 데이터를 저장해 기록을 비교**할 수 있게 함 (예: 지난주 대비 추이)
- 수치는 진단이 아닌 **자가 참고값** — 개인 맞춤 운동 강도의 기준으로 활용

### ③ 미니게임 — 서브(보조) 🎮
- 손목·손가락 운동 동작을 활용한 미니게임으로 **지루하지 않게 보조**
- 게임은 가이드를 대체하지 않는 **선택 콘텐츠** ("오늘은 재미있게 하고 싶은 날")
- 측정된 개인 ROM에 맞춰 난이도 자동 조정

### 공통: 루틴 형성 장치
- 오늘의 루틴 제안, 연속 달성(스트릭), 완료 기록 적립
- 기록이 쌓이고 변화가 보이면 스스로 계속하게 되는 선순환

## 5. 사용 흐름 (데일리 루틴)

```
접속 → 오늘의 루틴 제안
 → ① 숏폼 가이드 보고 따라 하기 (1~3분)
 → ② (주기적) ROM 측정 → 기록·추이 확인
 → ③ (선택) 미니게임으로 재미있게 한 번 더
 → 기록 적립 · 스트릭 → 내일 다시
```

## 6. 구현 계획

**기술 스택**
- 웹앱, **모듈 분리 구조**(HTML + CSS + JS 모듈) — 수정·확장·유지보수 용이
- MediaPipe Tasks Vision — 브라우저 온디바이스 손 인식 (영상 미저장, 좌표만 사용 → 개인정보 보호)
- Canvas 2D — 가이드 애니메이션·게임 렌더링
- localStorage — 측정·운동 기록 로컬 저장

**개발 단계**
| 단계 | 내용 |
|---|---|
| 1 | 앱 뼈대 — 홈(오늘의 루틴) + 가이드/측정/게임 3섹션 구조 |
| 2 | 손 인식 엔진 — 손목 각도·손가락 모양 실시간 인식 |
| 3 | **숏폼 운동 가이드 모듈** — 시범 애니메이션 + 따라하기 인식 체크 |
| 4 | **ROM 측정 + 기록 저장·비교** — 측정 → 저장 → 추이 확인 |
| 5 | 미니게임 (서브 콘텐츠) |
| 6 | 루틴 장치(오늘의 루틴·스트릭) 및 사용성 개선 |

## 7. 근거 자료 (문헌 — 전문성)

**문제의 규모 (국내 통계)**
- 손목터널증후군 진료인원 연평균 약 9% 증가, 2013년 기준 약 17.5만 명·총진료비 약 396억 원 (건강보험심사평가원) — 장시간 컴퓨터 작업·반복적 손 사용이 주요 위험 요인으로 지목됨

**게임화의 효과**
- 손목 골절 재활 무작위 대조 파일럿 연구: 게임화 그룹이 기존 재활 그룹보다 손 기능·기능적 독립성에서 유의하게 더 큰 개선 (Cureus, 2022)
- 근골격계 재활 문헌 리뷰: 게임 기반 운동이 기존 방식과 유사하거나 우수하며 운동 동기·삶의 질 개선 (Curr Rev Musculoskelet Med, 2022)
- 즉각적 피드백·보상이 참여도와 반복 횟수를 높이고, 참여도는 결과와 직접 상관

**운동 구성**
- 힘줄·신경 활주 운동이 병행 시 손목 불편에 긍정적 효과 (RCT 체계적 문헌고찰, 2015)
- 사무직 근로자 대상 8주 예방 운동 프로그램으로 손 기능 상태 유의 개선 (Int J Occup Med Environ Health, 2024)
- 임상 권장이 "짧게, 하루 여러 번"(힘줄활주 5~10회 × 하루 3~5회, 핀치 5~10초 유지) → **숏폼·마이크로 루틴 설계의 근거**

**측정 기술의 타당성**
- MediaPipe 기반 손 ROM 자동 측정이 기존 방법 대비 수용 가능한 수준으로 원격 손 평가에 적용 가능 (2023)
- MediaPipe 손 추적의 임상 검증 연구: 골드스탠다드 대비 높은 일관성 (2024)

**가동범위 기준**
- 일상생활 과제 대부분은 최대 가동범위의 약 70%로 수행 가능 (J Hand Surg, 1991) → 측정값 기반 개인화 목표 설계의 근거

## 8. 규제(의료법·의료기기법) 검토

- 식약처 「의료기기와 개인용 건강관리(웰니스) 제품 판단기준」상 본 앱은 **웰니스 제품**에 해당하도록 설계
  - 목적: 질병의 진단·치료가 아닌 **건강한 일반인의 피로 완화·습관 형성·가동성 유지**
  - 위해도: 가벼운 스트레칭 안내·자가 측정으로 낮음
- 스트레칭 가이드 제공(보고 따라 하기)은 일반 대중 대상 운동 정보 제공으로 의료행위가 아님
- 준수 원칙
  1. 진단·치료·처방 표현 금지 (측정값은 "참고", 판정 아님)
  2. 측정 결과와 운동 연결은 "제안" 수준 (판정+처방 구조 금지)
  3. 면책 문구: "본 서비스는 의료기기가 아니며 질병의 진단·치료 목적이 아닙니다. 통증이 있거나 질환이 의심되면 의사와 상담하세요."
  4. 각 가이드에 "통증 시 즉시 중단" 안내
- 가이드 자료는 자체 제작으로 저작권 문제 없음

## 10. 기대 효과

- 개인: 바쁜 일상 속에서 짧고 재미있게 손목 피로를 관리하는 **데일리 루틴** 형성
- 사회: 손목 피로의 조기 자가 관리 문화 확산 — 예방적 건강 관리의 접근성 향상
- 프로젝트: 문헌 근거에 기반한 **근거 기반 웰니스 설계** 사례

### 부록 — 예상 질문 대응

**Q. 스트레칭 가이드 제공이 의료행위 아닌가?**
A. 일반 대중 대상 운동 정보 제공은 의료행위(개인 상태 판단 후 처치 지시)가 아니며, 웰니스 판단기준의 "건강한 습관 유도" 영역입니다. 진단·처방 표현을 배제하고 면책 문구를 표기합니다.

**Q. 웹캠 측정이 정확한가?**
A. MediaPipe 기반 손 ROM 측정의 적용 가능성이 문헌으로 확인되어 있으며, 본 앱은 정밀 진단이 아닌 자가 참고·추이 비교 목적입니다.

**Q. 게임 앱인가?**
A. 아닙니다. 숏폼 운동 가이드와 측정·기록이 메인이고, 게임은 지속성을 높이는 보조 장치입니다. 게임화가 참여도와 기능 개선에 기여한다는 근거에 따라 채택했습니다.
