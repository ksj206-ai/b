# assets — 마스코트 이미지

홈 화면 마스코트(우주비행사 고양이) 이미지를 이 폴더에 넣어주세요.
투명 배경 PNG 권장(밤하늘 위에 자연스럽게 얹힘). 파일명은 아래 그대로.

| 파일명 | 상태 | 언제 표시되나 |
|---|---|---|
| `cat-idle.png` | 기본(평상시) | 한 손 인사 — 평소/진행 중 |
| `cat-happy.png` | 신남 | 오늘 루틴 6개 완주 시 |
| `cat-sad.png` | 시무룩 | 스트릭 끊김 / 며칠 만에 방문 |

- 표시 크기는 132×132 박스에 `object-fit:contain`(비율 유지, 정사각형 아니어도 됨).
- 코드는 이미 이 경로(`assets/cat-idle.png` 등)를 참조합니다 — 파일만 넣으면 동작.
